How to Set Up a Free MapTiler Account and Get Your API Key

This tutorial will walk you through the steps to create a free MapTiler account and retrieve your API key so you can activate the interactive map feature on the Cirque Electrique web app.

Step 1: Go to MapTiler

Navigate to https://www.maptiler.com in your web browser.

Step 2: Create an Account

  1. Click Sign In in the top right corner.
  2. Choose Sign up for free.
  3. You can create an account using:
    • Google login
    • GitHub login
    • Email and password (create a new account manually)

Once your account is created, you'll be redirected to your personal dashboard.

Step 3: Get Your API Key

  1. In the dashboard, click API Keys from the left-hand menu.
  2. If one doesn't already exist, click Create API Key.
  3. Give it a name like CirqueMap and click Create.
  4. Copy the generated key.
This is the key you'll paste into the map page of the Cirque Electrique site.

Step 4: Insert the Key into the Web App

  1. Open the Cirque Electrique web app.
  2. Navigate to the Map page.
  3. When prompted for a MapTiler API key, paste the key you copied.

Once entered, the map should load with full functionality. You only need to do this once per device/browser.

Notes:

  • The free plan includes generous usage limits for casual player activity.
  • Do not share your key publicly.
  • If you ever lose your key, you can regenerate it from your MapTiler dashboard.

That's it—you’re ready to explore Cirque Electrique's London of 1880 using your own interactive map layer.
